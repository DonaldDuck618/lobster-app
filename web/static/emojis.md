## 使用场景

### Web 端
```
• 页面加载：思考表情
• 任务完成：满意表情
• 错误提示：惊讶表情
• 成功提示：开心表情
```

### 小程序
```
• 启动页：开心表情
• 加载动画：思考表情
• 空状态：惊讶表情
• 完成页：满意表情
```

### APP
```
• 通知栏：开心表情
• 推送消息：惊讶表情
• 任务提醒：加油表情
• 会员特权：爱心表情
```

## 尺寸规范

### PNG 格式
```
• 16x16px - 网站 favicon
• 32x32px - 小程序头像
• 64x64px - 通知图标
• 128x128px - 启动页
• 256x256px - APP 图标
• 512x512px - 应用市场
• 1024x1024px - 高清展示
```

### SVG 格式
```
• 矢量格式
• 无限缩放
• 推荐用于 Web
```

## 文件命名

### 规范
```
logo-[表情]-[尺寸].[格式]

示例：
logo-happy-512.png
logo-thinking-256.png
logo-surprised-128.png
```
