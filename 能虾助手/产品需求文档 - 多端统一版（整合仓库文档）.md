## 二、多端架构设计（整合自 ARCHITECTURE.md）

### 2.1 整体架构图

```yaml
architecture:
  layers:
    - name: 用户层
      components:
        - 微信小程序
        - iOS App
        - Android App
        - Web (桌面/移动)
    
    - name: API 网关
      components:
        - REST API
        - WebSocket
        - 统一认证
    
    - name: 业务服务层
      components:
        - 认证服务 (手机号/微信登录)
        - 对话服务
        - 文件服务
        - 工具服务 (Excel/搜索/文案)
        - 会员服务
    
    - name: 数据层
      components:
        - PostgreSQL (主数据库)
        - Redis (缓存/会话)
        - 阿里云 OSS (文件存储)
    
    - name: AI 能力层
      components:
        - 阿里云百炼 (通义千问)
        - 智谱 GLM
        - Kimi
        - MiniMax
```

### 2.2 多端复用架构

```yaml
multi_platform:
  code_reuse_rate: 70-90%
  
  shared_layer:
    - API 调用层 (auth.js, chat.js, file.js)

  platform_specific:
    wechat_miniprogram:
      framework: uni-app
```

### 3.2 详细功能说明

#### 3.2.1 用户注册/登录

```yaml
auth:
  methods:
    - phone_sms:
        name: 手机号 + 验证码
        process:
          - 输入手机号
          - 获取验证码
          - 输入验证码
          - 设置密码（首次）
          - 登录成功
    
    - wechat:
        name: 微信一键登录
        platforms: [小程序，Web]
        process:
          - 点击微信登录
          - 授权获取用户信息
          - 绑定手机号（首次）
          - 登录成功
  
  api:
    register: POST /api/v1/auth/register
    login: POST /api/v1/auth/login
    send_code: POST /api/v1/auth/send-code
    wechat: POST /api/v1/auth/wechat
```

#### 3.2.2 对话界面

```yaml
chat_ui:
  style: 类微信聊天界面
  features:
    - 文字消息
    - 图片消息
    - 文件消息
    - 引用回复
    - 消息撤回（2 分钟内）
    - 快捷指令（7 个预设）
  
 快捷指令:
    - "帮我写周报"
    - "分析这个 Excel"
    - "搜索行业报告"
    - "写一封邮件"
    - "生成营销文案"
    - "图片 OCR"
    - "语音输入"
  
  api:
    send: POST /api/v1/chat/send
    sessions: GET /api/v1/chat/sessions
    session_detail: GET /api/v1/chat/{id}
    delete: DELETE /api/v1/chat/{id}
```

#### 3.2.3 Excel 分析

```yaml
excel_analysis:
  supported_formats: [.xlsx, .csv]
  max_size: 100MB
  
  process:
    - 上传文件
    - 自动识别结构
    - 生成数据摘要
    - 自动分析（趋势/对比/异常值）
    - 生成图表
    - 导出报告
  
  output:
    - 数据概览（行数、列数、关键字段）
    - 主要发现（3-5 条）
    - 图表（柱状图/折线图/饼图）
    - 导出格式（PDF/Word）
  
  api:
    upload: POST /api/v1/files/upload
    analyze: POST /api/v1/tools/excel
```

### 6.1 阿里云部署

```yaml
deployment:
  provider: 阿里云
  
  ecs:
    instance: ecs.c6.large (2 核 4G)
    count: 2 (负载均衡)
    os: Ubuntu 22.04
    runtime: Node.js + PM2
  
  rds:
    type: PostgreSQL / MySQL
    instance: pg.n2.medium.2c
    storage: 100GB SSD
    backup: 自动备份（7 天）
  
  redis:
    type: Redis 6
    instance: redis.master.small
    memory: 2GB
  
  oss:
    bucket: xiabot-files
    region: cn-hangzhou
    storage: 标准存储
    cdn: 开启
  
  slb:
    type: 应用型负载均衡
    listeners: [80, 443]
```
