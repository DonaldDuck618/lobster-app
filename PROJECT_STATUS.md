## 🔐 安全提醒

### GitHub Token

当前使用的 Token 已暴露在对话中，**强烈建议**：

1. **立即删除当前 Token**
   ```
   访问：https://github.com/settings/tokens
   删除：[你的 Token]
   ```

2. **创建新 Token**
   - 使用 Fine-grained Personal Access Token
   - 只授予 `lobster-app` 仓库权限
   - 权限：Contents (读写)、Issues (读写)

3. **使用环境变量**
   ```bash
   export GITHUB_TOKEN=your_new_token
   ```
