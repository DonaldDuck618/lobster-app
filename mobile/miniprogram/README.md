# 🦞 赚好多能虾助手 - 微信小程序快速启动指南

## 🎯 目标

**30 分钟内完成微信小程序首次运行**

---

## 📋 步骤 1：注册小程序账号 (5 分钟)

### 访问注册页面

```
https://mp.weixin.qq.com
```

### 注册流程

1. 点击右上角 **"立即注册"**
2. 选择 **"小程序"**
3. 填写邮箱、密码
4. 邮箱激活
5. 信息登记
   - **主体类型选择**: 
     - ✅ **个人** - 免费，无需营业执照（推荐个人开发者）
     - 企业 - 300 元/年，需要营业执照
6. 完成验证

### 获取 AppID

1. 登录小程序后台
2. 进入 **开发** -> **开发管理** -> **开发设置**
3. 找到 **"AppID(小程序 ID)"**
4. 复制 AppID（格式：`wx____________`）

---

## 📥 步骤 2：下载开发工具 (5 分钟)

### 微信开发者工具

**下载：** https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html

**选择版本：**
- macOS (Apple 芯片) - M1/M2/M3 Mac
- macOS (Intel 芯片) - Intel Mac
- Windows - Windows 电脑

**安装后：**
- 打开微信开发者工具
- 使用微信扫码登录

---

## ⚙️ 步骤 3：配置项目 (5 分钟)

### 3.1 修改 AppID

打开 `manifest.json`，找到 `mp-weixin` 配置：

```json
"mp-weixin": {
  "appid": "wx____________",  // 替换为你的 AppID
  ...
}
```

### 3.2 安装依赖

```bash
cd lobster-app/mobile
npm install
```

---

## 🚀 步骤 4：运行小程序 (5 分钟)

### 方式一：使用 HBuilderX（推荐）

1. **下载 HBuilderX**
   - https://www.dcloud.io/hbuilderx.html

2. **导入项目**
   - 打开 HBuilderX
   - 文件 -> 导入 -> 从本地目录导入
   - 选择 `lobster-app/mobile` 目录

3. **运行到小程序**
   - 工具栏：**运行** -> **运行到小程序模拟器** -> **微信开发者工具**
   - 自动打开微信开发者工具
   - 在模拟器中预览

### 方式二：使用命令行

```bash
cd lobster-app/mobile

# 开发模式
npm run dev:mp-weixin

# 构建
npm run build:mp-weixin
```

然后用微信开发者工具打开 `dist/dev/mp-weixin` 目录。

---

## 📤 步骤 5：上传小程序 (5 分钟)

### 5.1 构建生产版本

```bash
npm run build:mp-weixin
```

生成目录：`dist/build/mp-weixin`

### 5.2 上传代码

1. 打开微信开发者工具
2. 导入项目：`dist/build/mp-weixin`
3. 点击右上角 **"上传"**
4. 填写版本号（如 1.0.0）
5. 填写版本备注（如 "首次提交"）
6. 点击上传

---

## 📋 步骤 6：提交审核 (5 分钟)

### 小程序后台操作

1. 登录 https://mp.weixin.qq.com
2. 进入 **版本管理**
3. 找到刚上传的 **开发版**
4. 点击 **"提交审核"**

### 准备审核材料

**必填材料：**

1. **功能截图** (3-5 张)
   - 首页截图
   - 对话界面截图
   - 个人中心截图

2. **功能介绍** (20-200 字)
   ```
   赚好多能虾助手 是一款 AI 智能助手，可以帮助用户处理 Excel 数据、
   撰写周报日报、搜索行业报告、生成营销文案等。通过自然语言
   交互，让用户在微信中就能享受 AI 带来的办公提效体验。
   ```

3. **隐私政策链接**
   - 可以放在公众号文章
   - 或使用第三方服务生成

4. **用户协议链接**

5. **测试账号** (如需要登录)
   - 提供一个测试账号供审核人员使用

### 审核时间

- **通常**: 1-7 个工作日
- **加急**: 可当天审核（有特殊情况可申请）

---

## 🎉 步骤 7：发布上线

审核通过后：

1. 登录小程序后台
2. 进入 **版本管理**
3. 找到 **审核版**
4. 点击 **"发布"**
5. 用户即可搜索到你的小程序

## ⚠️ 注意事项

### 小程序命名

**命名规则：**
- 长度：4-30 个字符
- 不能与已有小程序重复
- 不能包含敏感词
- 不能侵犯他人商标

**建议名称：**
- ✅ 赚好多能虾助手
- ✅ 龙虾 AI 助手
- ✅ 龙虾办公助手

### 服务类目选择

**个人主体可选：**
- 工具 - 办公
- 工具 - 效率
- 教育 - 培训

**企业主体可选：**
- 更多类目（需要相应资质）

### 审核避坑指南

**常见拒绝原因：**

1. ❌ **功能不完整**
   - 有空白页面
   - 功能无法使用
   - 按钮点击无响应

2. ❌ **诱导分享**
   - 要求用户分享才能使用
   - 分享后才能解锁功能

3. ❌ **违规内容**
   - 色情、暴力、政治敏感
   - 虚假信息

4. ❌ **隐私问题**
   - 未提供隐私政策
   - 过度收集用户信息

5. ❌ **虚拟支付**
   - iOS 端禁止虚拟物品支付
   - 会员订阅在 iOS 端需隐藏

**解决方案：**
- ✅ 确保核心功能完整可用
- ✅ 完善隐私政策和用户协议
- ✅ 添加客服入口
- ✅ 添加关于我们页面
- ✅ iOS 端隐藏支付功能

## 🔧 配置域名

### 小程序后台配置

1. 登录小程序后台
2. **开发** -> **开发管理** -> **开发设置**
3. **服务器域名**

### 配置以下域名

| 类型 | 域名 | 说明 |
|------|------|------|
| **request 合法域名** | `https://api.lobster-app.com` | API 接口 |
| **socket 合法域名** | `wss://api.lobster-app.com` | WebSocket |
| **uploadFile 合法域名** | `https://api.lobster-app.com` | 文件上传 |
| **downloadFile 合法域名** | `https://api.lobster-app.com` | 文件下载 |

### 域名要求

- ✅ 必须使用 HTTPS/WSS
- ✅ 域名必须 ICP 备案
- ✅ 每月最多修改 5 次

## 🐛 常见问题

### Q1: 小程序无法调用 API？

**检查：**
1. 域名是否已配置
2. 是否使用 HTTPS
3. 域名是否已备案
4. 是否在小程序后台配置

### Q2: 包大小超限？

**解决方案：**
1. 启用分包加载
2. 压缩图片资源
3. 移除无用代码
4. 使用 CDN 加载大文件

### Q3: 审核被拒？

**处理流程：**
1. 查看拒绝原因
2. 针对性修改
3. 重新提交审核
4. 如有异议可申诉

### Q4: 个人主体不能做什么？

**限制：**
- ❌ 不能有虚拟支付
- ❌ 不能选择企业服务类目
- ❌ 不能开通小程序直播

**但办公工具类不受影响！**
