## 一、产品概述

### 1.1 产品定位

移动端 AI 智能助手，专注于帮助用户**真正完成工作任务**，而不仅仅是聊天。

### 1.2 核心价值

| 维度 | 说明 |
|------|------|
| **能干活** | 不只是聊天，能真正执行任务（Excel 分析、搜索报告、文案写作） |
| **零门槛** | 一键安装、开箱即用，无需配置、无需代码基础 |
| **全场景** | 覆盖办公、学习、生活全场景，一个 APP 搞定 |
| **可定制** | 支持自定义 Skill，满足个性化需求 |

### 1.3 目标用户

| 用户群体 | 占比 | 核心需求 | 付费意愿 |
|----------|------|----------|----------|
| 职场白领/新媒体运营 | 60% | Excel 分析、文案生成、周报撰写 | 极高 (19-39 元/月) |
| 小微企业/电商卖家 | 20% | 客服自动化、选品分析、订单对账 | 高 (企业版年付) |
| 学生/自由职业者 | 15% | 论文资料搜集、数据处理 | 中 (月付会员) |
| 技术爱好者 | 5% | 自定义 Skill、DIY Agent | 中 (专业版) |

## 二、功能需求

### 2.1 MVP 版本 (v1.0) - 2 个月

#### 核心功能

| 功能模块 | 优先级 | 描述 | 验收标准 |
|----------|--------|------|----------|
| **用户注册/登录** | P0 | 手机号 + 验证码登录 | 可正常注册登录 |
| **对话界面** | P0 | 类微信聊天界面 | 流畅、支持富文本 |
| **文件上传** | P0 | Excel、PDF、图片上传 | 支持 100MB 以内文件 |
| **Excel 分析** | P0 | 上传表格，自动分析 | 生成图表 + 文字报告 |
| **网络搜索** | P0 | 多源搜索 + 报告生成 | 返回 5-10 条结果 + 总结 |
| **文案写作** | P0 | 周报、邮件、营销文案 | 符合场景、可直接使用 |

#### 功能详细说明

**1. 用户注册/登录**
- 手机号 + 验证码
- 微信一键登录 (可选)
- 忘记密码功能

**2. 对话界面**
- 类似微信的聊天 UI
- 支持文字、图片、文件
- 支持引用回复
- 支持消息撤回 (2 分钟内)

**3. 文件上传**
- 支持格式：Excel (.xlsx/.csv)、PDF、图片
- 大小限制：100MB
- 上传进度显示
- 上传后自动解析

**4. Excel 分析**
- 自动识别表格结构
- 生成数据摘要 (行数、列数、关键字段)
- 自动分析 (趋势、对比、异常值)
- 生成图表 (柱状图、折线图、饼图)
- 导出分析报告 (PDF/Word)

**5. 网络搜索**
- 多源搜索 (Tavily、百度、360)
- 返回 5-10 条相关结果
- 自动生成摘要总结
- 标注信息来源
- 支持导出为报告

**6. 文案写作**
- 预设模板 (周报、邮件、营销文案、通知)
- 自定义输入 (主题、要点、语气)
- 一键生成
- 支持多版本选择
- 支持编辑修改

## 三、非功能需求

### 3.1 性能要求

| 指标 | 要求 |
|------|------|
| **对话响应** | < 2 秒 |
| **文件上传** | 10MB 文件 < 10 秒 |
| **Excel 分析** | 1000 行数据 < 30 秒 |
| **搜索报告** | < 15 秒 |
| **APP 启动** | 冷启动 < 3 秒 |
| **可用性** | 99.5% |

### 3.2 安全要求

- 用户数据加密存储
- 传输使用 HTTPS
- 敏感信息脱敏
- 内容安全审核 (接入阿里云/腾讯云)
- 符合 GDPR/个人信息保护法

### 3.3 兼容性要求

| 平台 | 最低版本 |
|------|----------|
| **iOS** | iOS 13+ |
| **Android** | Android 8.0+ |
| **屏幕** | 支持手机、平板 |

## 四、技术架构

### 4.1 整体架构

```
┌─────────────────┐         ┌─────────────────┐
│   移动端 APP     │         │   云侧 Gateway   │
│  (React Native) │ ◄─────► │  (OpenClaw)     │
│                 │  WS/API │                 │
│ • 对话 UI        │         │ • Agent 引擎     │
│ • 文件上传       │         │ • 工具链         │
│ • 用户管理       │         │ • Skill 生态     │
└─────────────────┘         └─────────────────┘
                                   │
                            ┌──────┴──────┐
                            │             │
                     ┌──────▼──────┐ ┌────▼────┐
                     │  大模型 API  │ │ 搜索 API │
                     │ 通义/智谱/Kimi│ │Tavily/百度│
                     └─────────────┘ └─────────┘
```

### 4.2 技术栈

| 模块 | 技术选型 | 备选方案 |
|------|----------|----------|
| **前端框架** | React Native / uniapp | Flutter |
| **UI 组件** | uView UI | Ant Design Mobile |
| **状态管理** | Pinia / Redux | Zustand |
| **云侧服务** | OpenClaw Gateway (Node.js) | 自研 |
| **数据库** | PostgreSQL + Redis | MySQL |
| **文件存储** | 阿里云 OSS | 腾讯云 COS |
| **消息推送** | 极光推送 | 个推 |
| **内容审核** | 阿里云内容安全 | 腾讯云内容安全 |
