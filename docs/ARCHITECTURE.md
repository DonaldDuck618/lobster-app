## 🎯 技术栈总览

```
┌─────────────────────────────────────────────────────────────────────────┐
│                          技术栈总览                                     │
└─────────────────────────────────────────────────────────────────────────┘

├─ 运行时：Node.js 22+
├─ 框架：Express.js
├─ 语言：TypeScript/JavaScript
├─ 进程管理：PM2
└─ API 文档：Swagger

数据库:
├─ 主数据库：PostgreSQL / MySQL
├─ 缓存：Redis
├─ 文件存储：Aliyun OSS
└─ ORM: Sequelize / TypeORM

第三方服务:
├─ 大模型：通义千问 / 智谱 GLM / Kimi
├─ 短信：阿里云短信
├─ 内容安全：阿里云内容安全
├─ 搜索：Tavily / 百度
└─ 支付：微信支付 / 支付宝

DevOps:
├─ 版本控制：Git + GitHub
├─ CI/CD: GitHub Actions
├─ 监控：云监控 + 日志服务
└─ 部署：阿里云 ECS
```
