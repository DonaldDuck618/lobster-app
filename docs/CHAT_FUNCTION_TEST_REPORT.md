## 🔍 问题排查

### 如果无法对话

**检查步骤:**
1. 确认已登录（左下角显示用户名）
2. 检查网络连接
3. 清除浏览器缓存
4. 更换浏览器测试

**常见错误:**
- 401 Unauthorized → 未登录，需要重新登录
- 500 Internal Server Error → 服务器错误，联系管理员
- Network Error → 网络问题，检查连接
